# DefaultAppearances 1.0.6

Renamed public package/addon naming to **DefaultAppearances**.

## Install

1. Extract the `DefaultAppearances` folder.
2. Put it inside your ArcheAge Classic `addons` folder.
3. Add `DefaultAppearances` to `addons.txt`.

## Notes

- Easy toggle on and off for default appearances.
- Button looks and placement comes from Unsafe Portals by Notuli.
