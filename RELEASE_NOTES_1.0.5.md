# Toggle Appearance 1.0.5

Initial public release.

## Features

- Easy toggle for default appearances.
- OFF and ON button states.
- ON state uses the 100-player display setting.
- Clean button styling and placement inspired by Unsafe Portals by Notuli.

## Install

Extract `ToggleAppearance` into your ArcheAge Classic `addons` folder and add `ToggleAppearance` to `addons.txt`.
